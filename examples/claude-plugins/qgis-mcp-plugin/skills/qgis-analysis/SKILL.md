# QGIS 空间分析

## 能力

此技能赋予你以下 QGIS 能力:

1. **数据加载**: 加载 Shapefile、GeoJSON、GPKG 等矢量数据和 GeoTIFF 等栅格数据
2. **空间查询**: 按属性、空间关系选择要素
3. **缓冲区分析**: 围绕点/线/面生成缓冲区
4. **叠加分析**: 交集、并集、差集等空间叠加操作
5. **地图渲染**: 生成专题地图并导出为 PDF/PNG
6. **坐标变换**: 在不同 CRS 之间转换

## 使用方式

当需要进行空间分析时,先在 QGIS 项目中加载数据,然后通过 mcp:qgis 工具调用具体的分析功能。分析完成后可以导出结果。
